NAME
--------------------------------------
Example 3


DESCRIPTION
--------------------------------------
Spectral simulations for the spin system consisting of the high-spin Fe(III) ion and the nitroxide. 


HOW TO RUN  
--------------------------------------
1) Open Terminal or Command Prompt.

2) Navigate to the root directory of the program:
   cd …/DipFit/source_code
   
3) Run the program by the following command:
   python main.py examples/example03_hs_iron(III)_nitroxide/config_ex03_1.cfg (simulates the dipolar spectrum vs xi)
   python main.py examples/example03_hs_iron(III)_nitroxide/config_ex03_2.cfg (simulates the dipolar spectrum for xi = 90°)
   python main.py examples/example03_hs_iron(III)_nitroxide/config_ex03_3.cfg (simulates the dipolar spectrum for xi = 0°)

4) The results of the DipFit analysis will be saved in the folder:
   /DipFit/source_code/examples/example03_hs_iron(III)_nitroxide/
